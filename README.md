# vanWeperenV_2026_EpiduralRTX
R code for Gene Ontology pathway enrichment (RITAN) of plasma differentially expressed proteins and dot-plot visualization, comparing epidural resiniferatoxin (cRTX+MI) vs vehicle (MI) infarcted pigs. Companion to van Weperen et al. DEPs provided by the UCLA Proteome Research Center; analysis and figure code only.
